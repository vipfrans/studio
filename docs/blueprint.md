# **App Name**: KoroneBet.xyz

## Core Features:

- Admin Panel & Robux Management: A developer 'Admin Panel' toggled by a sleek ⚒️ icon to add mock Robux for demonstration, complementing a glowing, animated Robux counter displaying the user's balance.
- Interactive Game Selection Grid: A high-end Game Selection Grid featuring unique, high-quality images for Rocket, Mines, Coinflip, and Cases, with smooth Framer Motion page transitions on selection.
- Rocket & Mines Game Simulations: Functional simulations for the Rocket game (multiplier graph, live active bets) and Mines game (clean 5x5 glass grid with animated reveals and adjustable bomb count).
- Coinflip & Cases Game Simulations: Functional simulations for the Coinflip game (high-fidelity 1v1 flipping animation) and Cases game (horizontal 'Case Opening' slider featuring legendary Roblox items).
- Live Winnings Feed: A vertical 'Live Winnings' feed, styled exactly like Bloxflip, displaying user avatar, username, game icon, and winnings in a vivid, Glowing Neon Green font.
- Community Jackpot & Daily Rewards Display: Visual elements including a progress bar for the 'Community Jackpot' and a glass-styled card for 'Daily Rewards' to enhance user engagement.

## Style Guidelines:

- Primary Color: Electric Purple (#C899FF), chosen for its vibrant and futuristic appeal, to highlight interactive elements and active states against the dark background.
- Background Color: Deep Midnight Black (#161317), providing a dark, immersive canvas with a subtle purple undertone to evoke a premium and futuristic OS aesthetic.
- Accent Color: Luminous Magenta (#FF99E6), used for subtle glowing borders, secondary highlights, and special effects, adding depth and complementing the electric purple.
- Live Winnings Text Color: Glowing Neon Green (#73FF73), specifically for displaying positive winning amounts, ensuring high visibility and an 'electric' feedback in the winnings feed.
- Headline Font: 'Space Grotesk' (sans-serif), for a technical, modern, and distinct look suitable for titles and prominent text, reflecting a futuristic OS.
- Body Font: 'Inter' (sans-serif), for clean, legible body text and UI labels, ensuring readability across the application while maintaining a contemporary feel.
- Minimalist, sleek, and glass-styled icons with subtle glowing violet outlines, consistent with the Glassmorphism theme, for all navigation, game representations, and UI elements.
- Ultra-modern Glassmorphism design applied throughout the UI, characterized by rounded corners, heavy background blurs (backdrop-filter), and subtle glowing violet borders. The layout will be modular and fully responsive for mobile devices, designed as a premium futuristic OS.
- A continuous, high-quality 'Snowing Particle Effect' gracefully falling across the entire screen. Includes smooth counting animations for the Robux counter, Framer Motion-powered page transitions, and dedicated, high-fidelity animations for each game (Mines reveals, Coinflip flipping, Cases slider).
